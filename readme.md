# tsconfig

> dhrv's shared [ts config](https://www.typescriptlang.org/docs/handbook/tsconfig-json.html) for common projects

## install

```sh
npm install --save-dev @0xdhrv/tsconfig
```

## usage

`tsconfig.json`

```json
{
 "extends": "@0xdhrv/tsconfig",
 "compilerOptions": {
  "outDir": "dist"
 }
}
```
